# Bitrix Mapper — подготовка данных к импорту в Битрикс 24

На входе: JSON-файл (`input/leads.json`) — массив объектов.

Поддерживаются два формата:
- **Битрикс 24** — ключи совпадают с названиями колонок
- **Яндекс.Карты** — определяется автоматически по ключам `indexNumber`, `parsedAt`, `phone_1`

На выходе: CSV-файл (`output/leads_ready.csv`) с разделителем `;` и кодировкой UTF-8 с BOM.

## Установка

Дополнительные зависимости не требуются — только **Python 3.8+** (стандартная библиотека).

## Запуск

**Командная строка:**
```bash
python3 main.py
