# 📞 Bitrix Mapper — обработчик лидов для Битрикс 24

Программа для менеджеров кол-центра: чистит CSV-экспорт из Битрикса, находит дубликаты и создаёт готовый Excel для обратного импорта.

## Что делает
- 📞 Телефоны: 8(999)123-45-67 → +79991234567
- 📧 Email: убирает пробелы, приводит к нижнему регистру
- 📅 Даты: приводит к формату ДД.ММ.ГГГГ
- 🔴 Дубликаты: находит повторы по телефону, email, компании (отдельный лист с красной заливкой)
- ⚠️ Ошибки: битые строки собираются на лист «Ошибки» с пояснением

## Как скачать
Нажми зелёную кнопку **Code** → **Download ZIP**. Распакуй архив.

## Как запустить

### Windows
1. Установи Python: [python.org](https://python.org) (галочка «Add Python to PATH»)
2. Установи зависимости: открой PowerShell в папке проекта и введи `pip install -r requirements.txt`
3. Запусти: дважды кликни `Запустить.pyw`

### Mac
1. Python уже установлен. Если нет: [python.org](https://python.org)
2. Установи зависимости: открой Терминал в папке проекта и введи `pip3 install -r requirements.txt`
3. Запусти: дважды кликни `Запустить.command`

## Как пользоваться
1. Выгрузи лиды из Битрикса: CRM → Лиды → Экспорт → CSV (разделитель «точка с запятой»)
2. Нажми «Обзор» и выбери CSV-файл
3. Нажми «Обработать»
4. Нажми «Открыть Excel» — готово!

## Состав проекта
- `app.py` — графическое приложение (Tkinter)
- `main.py` — консольная версия
- `mapper.py` — логика обработки и генерации Excel
- `config.py` — настройки (разделитель, кодировки, лимиты)
- `validators.py` — проверка телефонов, email, дат

## Требования
- Python 3.8+
- openpyxl (установится через requirements.txt)
