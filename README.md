# Bitrix Mapper — подготовка CSV к импорту в Битрикс 24
На входе: CSV-экспорт лидов из Битрикса
На выходе: Excel-файл output/leads_ready.xlsx с тремя листами
- Лиды — очищенные данные (телефоны +7, email lowercase, даты ДД.ММ.ГГГГ)
- Дубликаты — строки с повторами телефонов/email/компаний (красная заливка)
- Ошибки — строки, не прошедшие обработку

Установка: pip install -r requirements.txt

Разделитель CSV задаётся в config.py (`CSV_DELIMITER`), по умолчанию `;`.

При ошибках в строках программа завершится с кодом 1; Excel при этом всё равно сохраняется (перезаписывается).

Ограничение: обрабатываются только стандартные поля Битрикса (64 колонки); пользовательские поля (UF_*, UTM) в Excel не попадают.

Запуск: python3 main.py
